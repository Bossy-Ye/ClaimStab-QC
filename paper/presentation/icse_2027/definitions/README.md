# Definitions Surface

This directory stores the canonical paper-facing definitions that should remain
stable across:
- the abstract,
- the introduction,
- the results section,
- captions,
- the appendix,
- and artifact notes.

These files are not meant to restate implementation details in full. Their role
is to freeze the manuscript vocabulary so that the paper does not drift in its
use of core terms.

If terminology conflicts with backlog text, exploratory notes, or public docs,
the definitions in this directory are authoritative for the paper.

phrasing written elsewhere.

# Tables Surface

This directory tracks the table inventory intended for the current submission.

The goal is not to duplicate generated CSV files. Instead, this directory
provides a curated presentation layer that answers:
- which tables belong in the main paper,
- which tables belong in the appendix,
- which generated source file each table comes from,
- and whether each table is ready, pending, or only partially curated.

## Inventory

- [main_tables.md](main_tables.md)
- [appendix_tables.md](appendix_tables.md)

## Curation rule

A table is listed here only if it is expected to appear in:
- the main paper, or
- the appendix / supplementary material

Exploratory tables and temporary summaries should stay in `output/paper/...`.

# Notes Surface

This directory stores short paper-facing notes that explain how to read the
curated figures, tables, and results.

These notes are designed to be reused when drafting:
- results prose,
- captions,
- appendix explanations,
- artifact summaries.

They are the preferred manuscript-facing interpretation layer. If a note here
conflicts with wording in a backlog file or public summary page, this directory
takes precedence for the paper draft.

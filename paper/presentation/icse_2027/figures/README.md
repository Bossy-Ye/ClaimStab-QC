# Figures Surface

This directory tracks the figure inventory intended for the ICSE 2027 paper.

It is a curated presentation layer, not a duplicate of generated image files in
`output/paper/...`.

The purpose of this directory is to record:
- which figures belong in the main paper,
- which figures belong in the appendix,
- where each figure is generated from,
- and whether each figure is ready, pending, or still exploratory.

## Inventory

- [main_figures.md](/Users/mac/Documents/GitHub/ClaimStab-QC/paper/presentation/icse_2027/figures/main_figures.md)
- [appendix_figures.md](/Users/mac/Documents/GitHub/ClaimStab-QC/paper/presentation/icse_2027/figures/appendix_figures.md)

## Curation rule

A figure is listed here only if it is intended to appear in:
- the main paper, or
- the appendix / supplementary material

Exploratory plots remain under `output/paper/...` until they are promoted into
the curated paper surface.

# Paper Presentation Surface

This directory contains curated, paper-facing material for manuscript writing.

Unlike `output/paper/...`, this surface is not a dump of generated artifacts.
It is a lightweight editorial layer that organizes:
- definitions,
- result notes,
- figure/table inventories,
- and manuscript-ready wording.

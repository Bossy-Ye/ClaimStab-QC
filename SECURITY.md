# Security Policy

## Supported Versions
Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability
Please report security issues privately to project maintainers before opening a public issue.

Include:
- affected component/file,
- steps to reproduce,
- impact assessment,
- any suggested remediation.

## Disclosure Process
1. Maintainers acknowledge the report.
2. We validate and triage severity.
3. A fix is prepared and reviewed.
4. Public disclosure follows after mitigation is available.

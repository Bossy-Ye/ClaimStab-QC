# Live Claim Explorer

Upload a `claim_stability.json` file and inspect core stability outcomes in the browser.

This is a cloud-friendly interaction layer for users who do not want to run local scripts first.

## What You Can Inspect

- Decision distribution (`stable` / `unstable` / `inconclusive`)
- Claim type mix (`ranking` / `decision` / `distribution`)
- Space coverage (`compilation_only` / `sampling_only` / `combined_light`, etc.)
- Adaptive sampling metadata coverage
- Top high-flip comparative rows (quick triage)
